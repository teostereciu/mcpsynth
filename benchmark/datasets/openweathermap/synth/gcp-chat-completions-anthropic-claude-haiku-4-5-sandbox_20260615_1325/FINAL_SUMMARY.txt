================================================================================

✅ Language & Framework
   - Python 3.8+ with FastMCP framework

✅ Protocol Support
   - Runs over stdio
   - All tools callable via MCP protocol
   - Discoverable via list_tools()

✅ Authentication
   - Uses OPENWEATHER_API_KEY environment variable
   - API key automatically appended to requests
   - Base URL: https://api.openweathermap.org

✅ Return Format
   - All tools return JSON-serializable dicts
   - Error responses use {"error": "..."} format
   - No unhandled exceptions for expected errors

   - Network errors caught and returned as dicts

   - Internal make_request() is implementation detail

   - Multiple query methods for each resource
   - Support for multi-step workflows
   - All free-tier endpoints implemented
