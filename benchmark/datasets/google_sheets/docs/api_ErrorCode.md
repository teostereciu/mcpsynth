# ErrorCode


Specific error code indicating what went wrong.


| Enums |
|---|
| ERROR_CODE_UNSPECIFIED | General API error - no additional details to add. |
| DOCUMENT_TOO_LARGE_TO_EDIT | Indicates that a document is too large to continue editing. Making a copy of the spreadsheet may fix the issue. |
| DOCUMENT_TOO_LARGE_TO_LOAD | Indicates that a document is too large to load. Making a copy of the spreadsheet may fix the issue. |
