## 🚀 Quick Start

```bash
# 1. Install
pip install -r requirements.txt

# 2. Configure
export EBAY_APP_ID="your_app_id"
export EBAY_CERT_ID="your_cert_id"
export EBAY_REFRESH_TOKEN="your_refresh_token"
export EBAY_ENVIRONMENT="SANDBOX"

# 3. Run
python server.py
```
