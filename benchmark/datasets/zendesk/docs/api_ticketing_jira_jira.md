# Jira Integration API

## On this page

  * [Authentication](/api-reference/ticketing/jira/jira/#authentication)

The official [Jira integration service](https://www.zendesk.com/apps/support/jira/) by Zendesk provides a streamlined collaboration experience for your support and engineering teams by bringing Zendesk Support and Jira together. The integration service consists of a Zendesk Support app, a Jira app, and a middleware that provides backend functionalities and data storage for apps. The Jira Integration API allows further customization of the integration to fit your particular requirements.

## Authentication

The Jira Integration API is part of the Zendesk API. It allows an authenticated admin user to view and create links between your Jira and Zendesk instances. Authentication works as described in the [the Zendesk API introduction](/api-reference/introduction/security-and-auth/).
