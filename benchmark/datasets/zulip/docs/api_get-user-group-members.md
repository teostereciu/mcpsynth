# Get user group members
GET https://your-org.zulipchat.com/api/v1/user_groups/{user_group_id}/members
Get the members of auser group.
Changes: New in Zulip 6.0 (feature level 127).

## Usage examples
curl
- curl
The-ulineimplements HTTPBasicauthentication.
See theAuthorizationheaderdocumentation for how
to get those credentials for Zulip users and bots.

```
Authorization
```

```
curl -sSX GET -G https://your-org.zulipchat.com/api/v1/user_groups/38/members \
    -u EMAIL_ADDRESS:API_KEY \
    --data-urlencode direct_member_only=false
```

```
curl -sSX GET -G https://your-org.zulipchat.com/api/v1/user_groups/38/members \
    -u EMAIL_ADDRESS:API_KEY \
    --data-urlencode direct_member_only=false
```

## Parameters
user_group_idintegerrequired in path
The ID of the target user group.
direct_member_onlybooleanoptional
Whether to consider only the direct members of user group and not members
of its subgroups. Default isfalse.

## Response

#### Return values
- members:(integer)[]A list containing the user IDs of members of the user group.
