# Create a message reminder
POST https://your-org.zulipchat.com/api/v1/reminders
Schedule a reminder to be sent to the current user at the specified time. The reminder will link the relevant message.
Changes: New in Zulip 11.0 (feature level 381).

## Usage examples
curl
- curl
The-ulineimplements HTTPBasicauthentication.
See theAuthorizationheaderdocumentation for how
to get those credentials for Zulip users and bots.

```
Authorization
```

```
curl -sSX POST https://your-org.zulipchat.com/api/v1/reminders \
    -u EMAIL_ADDRESS:API_KEY \
    --data-urlencode message_id=1 \
    --data-urlencode scheduled_delivery_timestamp=5681662420 \
    --data-urlencode 'note=This is a reminder note.'
```

```
curl -sSX POST https://your-org.zulipchat.com/api/v1/reminders \
    -u EMAIL_ADDRESS:API_KEY \
    --data-urlencode message_id=1 \
    --data-urlencode scheduled_delivery_timestamp=5681662420 \
    --data-urlencode 'note=This is a reminder note.'
```

## Parameters
message_idintegeroptional
The ID of the previously sent message to reference in the reminder message.
scheduled_delivery_timestampintegeroptional
The UNIX timestamp for when the reminder will be sent,
in UTC seconds.
notestringoptional

```
"This is a reminder note."
```
A note associated with the reminder shown in the Notification Bot message.
Changes: New in Zulip 11.0 (feature level 415).

## Response

#### Return values
- reminder_id:integerUnique ID of the scheduled message reminder.
