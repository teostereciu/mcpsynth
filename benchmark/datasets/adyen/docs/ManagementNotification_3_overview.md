# Management Webhooks
Adyen uses webhooks to inform your system about events that happen with your Adyen company and merchant accounts, stores, payment terminals, and payment methods when usingManagement API.
When an event occurs, Adyen makes an HTTP POST request to a URL on your server and includes the details of the event in the request body.
SeeWebhooksfor more information.
