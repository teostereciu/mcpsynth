# Get Shop Video Performance Overview

*Source: https://partner.tiktokshop.com/docv2/page/get-shop-video-performance-overview-202509*

---

# Path: /analytics/202509/shop_videos/overview_performance
# Method: [GET]
# Function Description
Returns overall performance metrics for all videos under a shop.

---
