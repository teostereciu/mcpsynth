# Get Shop Video Performance List

*Source: https://partner.tiktokshop.com/docv2/page/get-shop-video-performance-list-202509*

---

# Path: /analytics/202509/shop_videos/performance
# Method: [GET]
# Function Description
Returns a list of videos and associated metrics for a shop.
