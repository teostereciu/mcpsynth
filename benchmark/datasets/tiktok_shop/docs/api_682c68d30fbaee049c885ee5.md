# **本規約の概要**
要約すると、このグローバル：TikTok Shopパートナーセンター利用規約は、貴社／貴殿とTikTokとの間の契約であり、貴社／貴殿によるTikTok Shopパートナーセンタープラットフォームへのアクセス及びその使用に適用されるものです。特に、本規約は、
· TikTok Shopパートナーセンタープラットフォーム上で販売者、クリエイター、ホスト又はその他の第三者（「サブスクライバー」と称します。）に対して貴社／貴殿のサービスをマーケティング、宣伝及び／又は提供するために充足する必要がある要件の概要を記載しています（第3条（*パートナーサービス*）及び第4条（*貴社／貴殿の本アカウント*））。
· パートナーとサブスクライバー間のいずれの契約についても、TikTokが当事者ではないことを定めています（第3.5条及び第3.9条）。 
· パートナーセンター商品にアクセスし、これを利用する際に、貴社／貴殿が許容されること及び禁止されることに関するいくつかのルールを定めています（第7条（*貴社／貴殿によるパートナーセンター商品へのアクセス及びその利用*））*。*
· パートナーセンタープラットフォーム上のパートナーコンテンツに対する一部の権利がTikTokに付与される旨を定めています（第9条（*知的財産権*））。
· 貴社／貴殿がTikTokから受領する、貴社／貴殿のパートナーセンター商品の有効性又はパフォーマンスを詳述したデータをどのように使用できるかに関する条件を定めています（第10条（*TikTokデータ*））。
· TikTokが貴社／貴殿との関係を制限、停止又は終了する方法（第12条（*停止及び終了*））及び終了の結果（第13条（*終了の結果*））を定めています。
· その他当社が責任を負う損失を限定しています（第14条（*免責*）及び第16条（*責任の制限*））。
貴社／貴殿が本規約に基づきTikTokと争う場合に従うべき手順を定めています（第18条（*雑則*）及び貴社／貴殿が居住している場所により適用される可能性のある法域別個別条項が定められた附属書類1をご参照ください。）。
