# Get GMV Trend Performances

*Source: https://partner.tiktokshop.com/docv2/page/get-gmv-trend-performances-202502*

---

# Path: /analytics/202502/live_rooms/{live_room_id}/gmv_trend_performances
# Method: [GET]
# Function Description
This API gets the trend points of gmv related information
