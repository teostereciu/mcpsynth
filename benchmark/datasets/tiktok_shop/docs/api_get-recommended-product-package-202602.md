# Get Recommended Product Package

*Source: https://partner.tiktokshop.com/docv2/page/get-recommended-product-package-202602*

# Path: /product/202602/packages/recommend
# Method: [POST]
# Function Description
Retrieve a list of product recommended packages. (Only available in BR and MX)
