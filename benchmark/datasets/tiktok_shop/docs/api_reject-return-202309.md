# Reject Return

*Source: https://partner.tiktokshop.com/docv2/page/reject-return-202309*

---

# Path: /return_refund/202309/returns/{return_id}/reject
# Method: [POST]
# Function Description
Use this API to reject a buyer's return or refund request.
