# Get Interactive Trend Performances

*Source: https://partner.tiktokshop.com/docv2/page/get-interactive-trend-performances-202502*

---

# Path: /analytics/202502/live_rooms/{live_room_id}/interactive_trend_performances
# Method: [GET]
# Function Description
This API gets interactive performance trend data.

---
