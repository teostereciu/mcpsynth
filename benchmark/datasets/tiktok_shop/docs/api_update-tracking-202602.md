# Update Tracking

*Source: https://partner.tiktokshop.com/docv2/page/update-tracking-202602*

---

# Path: /fbt/202602/inbound_orders/tracking_update
# Method: [POST]
# Function Description
This API manages carton-level tracking information for inbound orders. It utilizes upsert logic to create new tracking records or overwrite existing entries as required. Only orders which have not yet been marked as "Delivered" can be updated.
