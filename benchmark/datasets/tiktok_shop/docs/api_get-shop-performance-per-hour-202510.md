# Get Shop Performance Per Hour

*Source: https://partner.tiktokshop.com/docv2/page/get-shop-performance-per-hour-202510*

---

# Path: /analytics/202510/shop/performance/{date}/performance_per_hour
# Method: [GET]
# Function Description
Daily performance per hour detail, within 30 days.
Including today.
