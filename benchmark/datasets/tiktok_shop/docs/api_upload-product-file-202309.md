# Upload Product File

*Source: https://partner.tiktokshop.com/docv2/page/upload-product-file-202309*

---

# Path: /product/202309/files/upload
# Method: [POST]
# Function Description
Upload non-image files, such as PDF or video to TikTok Shop.
Use this API when you need to add videos to your product to improve the shopping experience, or submit certifications or reports to meet TikTok Shop requirements for listing restricted products.
**Note**: You must store the response body to retrieve the ID or URL required to associate the file with a product during product creation or editing.

# Request Body Parameters
| Properties | Type | Require | Sample | Properties description |
| --- | --- | --- | --- | --- |
| data | file | Y |  | The local file to be uploaded. |
**Note**：

* Supported formats: PDF, MP4, MOV, MKV, WMV, WEBM, AVI, 3GP, FLV, MPEG
* Max PDF size: 20 MB
* Max video size: 100 MB
* Video aspect ratio: 9:16 to 16:9

**Recommendations for product videos**:

* Aspect ratio: 1:1
* Resolution: HD 720p or higher
* Duration: 20 - 60 seconds
   |
   name |string |Y |SNI.pdf |The name of the file, including the file extension (e.g. `certification.pdf`)

**Note**:

* Do not use additional periods, except the one preceding the file extension.
* Do not begin the name with symbols.
* Do not include spaces. |
