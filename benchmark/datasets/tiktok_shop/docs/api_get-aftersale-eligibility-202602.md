# Get Aftersale Eligibility

*Source: https://partner.tiktokshop.com/docv2/page/get-aftersale-eligibility-202602*

---

# Path: /return_refund/202602/orders/{order_id}/aftersale_eligibility
# Method: [GET]
# Function Description
Use this API to check eligible aftersale solutions for an order, including whether the seller or buyer can initiate a refund, return, or cancel a specific order.
