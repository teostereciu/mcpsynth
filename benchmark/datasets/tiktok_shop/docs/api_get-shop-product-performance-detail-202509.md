# Get Shop Product Performance Detail

*Source: https://partner.tiktokshop.com/docv2/page/get-shop-product-performance-detail-202509*

---

# Path: /analytics/202509/shop_products/{product_id}/performance
# Method: [GET]
# Function Description
Return performance detailed metrics for a product.
