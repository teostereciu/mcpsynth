# Deprecating TikTok Shop API V1

*Source: https://partner.tiktokshop.com/docv2/page/deprecating-tts-api-v1*

---

To better serve our global developer community, **TikTok Shop Open Platform** is committed to building a high-performance, stable, and industry-compliant API system. Currently, we offer two main versions of our Open API:

* **V1 (Legacy version)**: version numbers below 202309;
* **V2 (Standard version)**: version numbers 202309 and above.

The V1 API was introduced in the early stages of our platform. To enhance the developer experience and improve integration compatibility, we launched the V2 version in **Q3 2023**, following modern industry standards.
## V1 API Deprecation Timeline
As part of our ongoing efforts to improve the platform, TikTok Shop will **gradually deprecate the V1 (Legacy) API** and fully shift to supporting and optimizing the V2 version. The deprecation will follow the timeline below:

* **Starting from July 1, 2025**, we will begin **phased deprecation of V1 APIs**;
* **By December 31, 2025**, all **V1 APIs will be fully sunset**.

During the phased deprecation period, the platform will gradually apply **request throttling policy** to the APIs scheduled for sunset and **include clear error messages** in the response to guide developers toward timely migration to the latest version.


In parallel, we will also establish a standardized API lifecycle management mechanism to ensure better control and predictability across the full lifecycle - from release and maintenance to deprecation. This will provide developers with a clearer, more efficient, and reliable integration experience.
## Call to Action for Developers
We strongly encourage all partners to **plan ahead for V2 migration**. TikTok Shop will provide a comprehensive **migration guide** and **technical support** to ensure a smooth transition and continued access to core e-commerce capabilities. For details, please visit [Migration Guide](upgrading-to-api-version-202309).
