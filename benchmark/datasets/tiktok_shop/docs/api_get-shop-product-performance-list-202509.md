# Get Shop Product Performance List

*Source: https://partner.tiktokshop.com/docv2/page/get-shop-product-performance-list-202509*

# Path: /analytics/202509/shop_products/performance
# Method: [GET]
# Function Description
Returns a list of product performance overview metrics.
