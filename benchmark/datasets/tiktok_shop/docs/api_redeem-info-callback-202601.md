# Redeem Info Callback

*Source: https://partner.tiktokshop.com/docv2/page/redeem-info-callback-202601*

---

# Path: /fulfillment/202601/redeem_info/callback
# Method: [POST]
# Function Description
Send the redeem info related to the order to the TTS to complete the fulfillment.
