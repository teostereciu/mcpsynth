# For SEA market: Product category expansion cutoff date is January 12, 2026

*Source: https://partner.tiktokshop.com/docv2/page/ezoyrtmn*

---

# What is changing?
On August 21, 2025, we introduced the product category expansion in SEA market. Please check this link: https://partner.tiktokshop.com/docv2/page/npb6fvbh
Now we announce that the cutoff date for this product category expansion is January 12, 2026. After this date, use **`category_version=V2`** parameter for all category trees when listing new products or editing products. 
This applies to the following Product APIs:
– [POST Create Product](https://partner.tiktokshop.com/docv2/page/create-product-202309)
– [PUT Edit Product](https://partner.tiktokshop.com/docv2/page/edit-product-202309)
# Which markets are affected?
The updates of the requirements apply to the Local to Local and Cross Border sellers for Singapore, Philippines, Vietnam, Thailand, Malaysia and Indonesia markets.
# Who is affected?
Developers with applications that use Product APIs listed above.
# Which version is applicable?
The updates of the requirements apply to all versions of the related APIs.
# What action is required?
Developers need to use **`category_version=V2`** for all category trees when listing new products or editing products.
