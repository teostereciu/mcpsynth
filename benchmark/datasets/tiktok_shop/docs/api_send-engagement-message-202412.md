# Send Engagement Message

*Source: https://partner.tiktokshop.com/docv2/page/send-engagement-message-202412*

---

# Path: /customer_engagement/202412/messages
# Method: [POST]
# Function Description
Send messages to specific customers for a particular engagement task.
