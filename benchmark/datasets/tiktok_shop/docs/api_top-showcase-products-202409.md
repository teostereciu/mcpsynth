# Top Showcase Products

*Source: https://partner.tiktokshop.com/docv2/page/top-showcase-products-202409*

---

# Path: /affiliate_creator/202409/showcases/products/top
# Method: [POST]
# Function Description
Use this API to move products to the top in a creator's showcase.
The platform will return the error code and error message if the pinning operation fails.

---

| Properties | Type | Require | Sample | Properties description |

| product_ids | []string | Y | ["12390753231","7102893481290"] | The product IDs to move to the top in a creator's showcase. If multiple products are provided, they will display according to the order passed in this parameter. |
