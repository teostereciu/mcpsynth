# Get Traffic Performances

*Source: https://partner.tiktokshop.com/docv2/page/get-traffic-performances-202502*

---

# Path: /analytics/202502/live_rooms/{live_room_id}/traffic_performances
# Method: [GET]
# Function Description
This API gets the traffic performances of live streaming for distribution analysis
