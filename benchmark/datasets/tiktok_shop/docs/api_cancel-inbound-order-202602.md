# Cancel Inbound Order

*Source: https://partner.tiktokshop.com/docv2/page/cancel-inbound-order-202602*

---

# Path: /fbt/202602/inbound_orders/cancel
# Method: [POST]
# Function Description
This API is used to cancel an inbound order prior to warehouse arrival, preventing further processing and updating the order status.
