# Product Review Removal

*Source: https://partner.tiktokshop.com/docv2/page/product-review-removal-202602*

---

# Path: /review_rating/202602/product_reviews/remove
# Method: [DELETE]
# Function Description
Remove product reviews from TikTok Shop due to external partner moderation
**Note**

* Only whitelisted partners can perform this removal action
* Only the reviews imported from the same partner can be removed
* Removed reviews will not be displayed on Seller Center nor TikTok Shop app
