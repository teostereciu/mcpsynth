# Get Product Stats

*Source: https://partner.tiktokshop.com/docv2/page/get-product-stats-202502*

---

# Path: /analytics/202502/live_rooms/{live_room_id}/product_stats
# Method: [GET]
# Function Description
This API gets the product list of live room

---
