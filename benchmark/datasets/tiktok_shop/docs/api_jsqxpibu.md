# For SEA market: Configure "Cancellation status change" webhook to receive cancellation message

*Source: https://partner.tiktokshop.com/docv2/page/jsqxpibu*

---

Since TikTok Shop is proposing to allow buyers to cancel eligible orders between the status Ready To Ship (RTS) and Transit To Ship (TTS), please configure the "[Cancellation status change](https://partner.tiktokshop.com/docv2/page/11-cancellation-status-change)" webhook to receive cancellation messages. If sellers receive such a cancellation message, they need to takeout the parcel and do not ship it.
This webhook's related links are as below:

* "Cancellation status change" webhook: https://partner.tiktokshop.com/docv2/page/11-cancellation-status-change
* Webhook overview: https://partner.tiktokshop.com/docv2/page/tts-webhooks-overview
* Webhook configuration guide: https://partner.tiktokshop.com/docv2/page/configuration-guide

If you have configured this webhook, please ignore this change log.
# Which markets are affected?
The updates of the requirements apply to the Local to Local sellers for Singapore, Philippines, Vietnam, Thailand, Malaysia and Indonesia markets.
# Who is affected?
Developers with applications that use Cancellation, Return or Refund related functions.
# Which version is applicable?
The updates of the requirements apply to all versions of the related webhook.
# What action is required?
Developers need to config "[Cancellation status change](https://partner.tiktokshop.com/docv2/page/11-cancellation-status-change)" webhook to receive cancellation messages. If developers have configured this webhook, please ignore this change log.
