# Get Creator Applicable Sample Label

*Source: https://partner.tiktokshop.com/docv2/page/get-creator-applicable-sample-label-202412*

---

# Path: /affiliate_creator/202412/samples/labels
# Method: [GET]
# Function Description
Check if a creator can apply for a sample of a specific product.

---
