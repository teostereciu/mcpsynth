# Get Shop Video Product Performance List

*Source: https://partner.tiktokshop.com/docv2/page/get-shop-video-product-performance-list-202509*

---

# Path: /analytics/202509/shop_videos/{video_id}/products/performance
# Method: [GET]
# Function Description
Returns performance metrics for a list of products promoted in a given video.
