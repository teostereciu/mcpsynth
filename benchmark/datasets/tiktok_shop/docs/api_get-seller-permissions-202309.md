# Get Seller Permissions

*Source: https://partner.tiktokshop.com/docv2/page/get-seller-permissions-202309*

---

# Path: /seller/202309/permissions
# Method: [GET]
# Function Description
Retrieves the cross-border operations that a cross-border seller is permitted to perform.
You can use this API prior to listing products to check whether a seller has the ability to list global products.
Target seller: Cross-border sellers

| Properties | Type | Sample | Properties description |
| --- | --- | --- | --- |
| code | int | 0 | The success or failure status code returned in API response. |
| message | string | Success | The success or failure messages returned in API response. Reasons of failure will be described in the message. |

| data | object |  | Specific return information |
| ^permissions | []string | ["MANAGE_GLOBAL_PRODUCT"] | The list of cross-border operations that the seller is permitted to perform. |
| Possible values: |  |  |  |

* MANAGE_GLOBAL_PRODUCT: Indicates the seller is permitted to manage global products listed in TikTok Shops across multiple countries.
   If this is empty, it means the seller does not have permission to conduct cross-border operations. |
