# New affiliate messaging APIs

*Source: https://partner.tiktokshop.com/docv2/page/new-affiliate-messaging-apis*

---

## Summary
This update centers around managing Affiliate Messaging, releasing 6 new endpoints.
## What is changing
As we continue to refine and expand our Creator Collaboration and Affiliate APIs, our goal is to empower ISVs with the tools they need to develop impactful and innovative solutions for TikTok Shop Sellers. Here are the newly released endpoints:

* Get Message in the Conversation
* Get Conversation List
* Send Message
* Create Conversation with Creator
* Mark Conversation Read
* Get Latest Unread Messages

## What action is required
If you are approved to use these APIs, we hope this improves your integration experience and can build functionality that is more native to your app.
