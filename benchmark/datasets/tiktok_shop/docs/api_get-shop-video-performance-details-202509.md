# Get Shop Video Performance Details

*Source: https://partner.tiktokshop.com/docv2/page/get-shop-video-performance-details-202509*

---

# Path: /analytics/202509/shop_videos/{video_id}/performance
# Method: [GET]
# Function Description
Returns detailed performance metrics for a (requested) video.

---
