# (59) Shoppable Video Precheck Tasks Result

*Source: https://partner.tiktokshop.com/docv2/page/arpnre1g*

---

## Trigger scenario
The Video Precheck Result webhook is triggered when the Precheck Video Content API is called, and provides the results of the video’s quality assessment and violation checks.
## Data business parameters
| **Parameter name** | **Data Type** | **Sample** | **Description** |
| --- | --- | --- | --- |
| type | int | `59` | The ID of the webhook type. |
| tts_notification_id | string | `"7327112393057389710"` | The ID of the notification. |
| creator_open_id | string | `"VIyStQAAAADCBQ40s4TzODDSEIW-bM5O9cod3vK8OytW8m-bnBYlXw"` | The open_id of the creator. To get the value, see [Authorization overview](authorization-overview-202407). |
| timestamp | int | `1644412885` | The UNIX timestamp when this webhook was triggered. |
| data | object |  |  |
| └task_id | string | `"7493990579714178574"` | ID of precheck task generated by TikTok Shop. |
| └violation_check_result | object |  |  |
| └└status | string | `SUCCESS` | Possible enumerations:  <br> `SUCCESS` <br> `FAIL` <br> `PROCESSING` |
| └└issues | []object |  |  |
| └└└risk | string | `"HIGH"` | Risk type. |
| └└└suggestions | string | `"Your video may include unoriginal content. Creating original content is essential for standing out from the crowd."` | Suggestions on how to fix the risk. |
| └good_quality_check_result | object |  |  |
| └└status | string | `FAIL` | Possible enumerations:  <br> `SUCCESS` <br> `FAIL` <br> `PROCESSING` |
| └└issues | []object |  |  |
| └└└code | string | `"LOW RESOLUTION"` |  Quality check code. |
| └└└suggestions | string | `"Your video may include unoriginal content. Creating original content is essential for standing out from the crowd."` | How to improve video quality. |
