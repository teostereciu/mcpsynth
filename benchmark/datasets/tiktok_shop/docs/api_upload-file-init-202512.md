# Upload File Init

*Source: https://partner.tiktokshop.com/docv2/page/upload-file-init-202512*

---

# Path: /open/202512/file/init
# Method: [POST]
# Function Description
To upload a large file to Tiktok Shop (e.g., a video), follow these three steps:

1. Initialize the upload session for the video.
2. Upload the video file to the TTS server. Files smaller than 5 MB must be uploaded as a single, non‑chunked request.
3. Call the API corresponding to the target path and pass the ResourceId returned in step 2 as a parameter.

For more detailed content, refer to the document:https://partner.tiktokshop.com/docv2/page/wfi3nz36
