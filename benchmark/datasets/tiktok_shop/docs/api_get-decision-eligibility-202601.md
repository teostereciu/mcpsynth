# Get Decision Eligibility

*Source: https://partner.tiktokshop.com/docv2/page/get-decision-eligibility-202601*

# Path: /return_refund/202601/decision_eligibility
# Method: [GET]
# Function Description
Use this API to check eligible decision for an return or cancellation, including whether the seller can approve, reject, direct_refund a specific return or cancellation request, this api also can response available reject reason when seller can execute reject decisio
