# Hazmat Deadline Extended to April 15

*Source: https://partner.tiktokshop.com/docv2/page/hazmat-deadline-extended-to-april-15*

---

## What is changing?
The previous deadline of hazmat requirements have been extended from 2025-03-05 to **2025-04-15**. Developers are encouraged to make changes prior to this date.
## Which markets and versions are affected?
All US and UK local sellers are impacted by this change. The change is applicable to all API versions.
