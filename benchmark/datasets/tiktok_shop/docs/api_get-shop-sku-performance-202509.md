# Get Shop SKU Performance

*Source: https://partner.tiktokshop.com/docv2/page/get-shop-sku-performance-202509*

# Path: /analytics/202509/shop_skus/{sku_id}/performance
# Method: [GET]
# Function Description
Return SKU performance metrics.
