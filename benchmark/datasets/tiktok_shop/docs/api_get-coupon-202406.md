# Get Coupon 

*Source: https://partner.tiktokshop.com/docv2/page/get-coupon-202406*

---

# Path: /promotion/202406/coupons/{coupon_id}
# Method: [GET]
# Function Description
Use this API to get the full details of a coupon matching the {coupon_id}.
