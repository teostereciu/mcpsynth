# Get Live Core Stats

*Source: https://partner.tiktokshop.com/docv2/page/get-live-core-stats-202502*

---

# Path: /analytics/202502/live_rooms/{live_room_id}/core_stats
# Method: [GET]
# Function Description
This API returns the core stats of live detail, e.g. GMV.

---
