# Get Shop SKU Performance List

*Source: https://partner.tiktokshop.com/docv2/page/get-shop-sku-performance-list-202509*

# Path: /analytics/202509/shop_skus/performance
# Method: [GET]
# Function Description
Returns a list of SKU performance metrics.
