# Get Shoppable Video Status

*Source: https://partner.tiktokshop.com/docv2/page/get-shoppable-video-status-202509*

---

# Path: /affiliate_creator/202509/videos/{video_id}/status
# Method: [GET]
# Function Description
Use this API to get shoppable video posting results.

---
