# Upload Product Image and Upload Product File support more file formats

*Source: https://partner.tiktokshop.com/docv2/page/upload-product-image-and-upload-product-file-support-more-file-formats*

---

## Summary
File formats supported by Upload Product Image and Upload Product File have been expanded.
## Which markets and versions are affected?
This change applies to **v202309** and affects all markets and sellers.
## API Changes
### Upload Product Image
We now support WEBP, HEIC, and BMP in addition to the previously supported JPG, JPEG, and PNG formats.
### Upload Product File
For video formats, we support MP4, MOV, MKV, WMV, WEBM, AVI, 3GP, FLV, and MPEG, in addition to the previously supported MP4 format.
