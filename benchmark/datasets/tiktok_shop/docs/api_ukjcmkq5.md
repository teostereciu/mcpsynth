# [Important] New Analytics API Supports Data Drill Down & Deprecation of Old Versions

*Source: https://partner.tiktokshop.com/docv2/page/ukjcmkq5*

---

# Summary
The existing **Analytics APIs** have a **new** version for `202509`, with the primary objective of aligning the data caliber with Seller Center Data Compass. Additionally, 3 new APIs have been added to provide data granularity to help merchants and influencers review, consolidate, and iterate content (video + live) production processes. 

Details of the new API endpoints and functions are listed below.


---


# Impact
| Impacted market(s) | * All markets - local and cross-border |
| --- | --- |
| Impacted version(s) | * 202509 (and later) |
## Important Note
Analytic API Versions `202406` and `202409` will be **deprecated** and offline in **April 2026**
