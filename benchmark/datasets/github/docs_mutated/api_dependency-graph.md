# REST API endpoints for the dependency graph
Use the REST API to view dependency changes and their security impact on your repository.
- REST API endpoints for dependency reviewGet a diff of the dependencies between commits
- REST API endpoints for dependency submissionCreate a snapshot of dependencies for a repository
- REST API endpoints for software bill of materials (SBOM)Export a software bill of materials (SBOM) for a repository.
- Get a diff of the dependencies between commits
- Create a snapshot of dependencies for a repository
- Export a software bill of materials (SBOM) for a repository.
